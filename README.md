# Kore (1.0.4)

Kore is a forge 1.8.9 mod, kore goal is to automate simple skyblock annoying and repetitive tasks.

![Powered by OneConfig](https://polyfrost.org/img/cozy_vector.svg)
## Features

- Render 
  - Watermark
- Player
  - Auto Harp
  - Auto Experiments
- Other
  - DevMode (Debug)
  - Mod Hider
